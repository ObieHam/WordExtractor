# Vocabulary Builder

An intelligent vocabulary building app that extracts words from PDFs and text files, fetches definitions from Merriam-Webster, and helps you build your personal vocabulary database.

## Features

- Upload PDFs or text files
- Automatic word extraction and filtering
- Merriam-Webster dictionary integration
- Audio pronunciations
- Custom image and audio uploads
- CSV export

## Tech Stack

- Frontend: React + Vite + Tailwind CSS
- Backend: Vercel Serverless Functions
- Database: Vercel Postgres
- Storage: Vercel Blob Storage
- API: Merriam-Webster Dictionary API
